判断代码：
int tadd_ok(int x, int y) {
    int s = x + y;
    // 正溢出：x>0, y>0, s<0
    // 负溢出：x<0, y<0, s>=0
    int pos_overflow = (x > 0) && (y > 0) && (s < 0);
    int neg_overflow = (x < 0) && (y < 0) && (s >= 0);
    return !pos_overflow && !neg_overflow;
}

右移k位 = 除以2^k 然后**向下取整（floor）**
